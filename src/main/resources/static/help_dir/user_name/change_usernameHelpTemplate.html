Изменение сетевого имени пользователя

 

1. Для изменения имени пользователя в сети войдите в личный кабинет и выберите пункт меню Анкета.

 

2. На странице Анкета выберите пункт меню Правка анкеты

 

3. На странице Правка анкеты в пункте "Имя в сети (nick)" в текстовом поле "Новое значение" введите новое имя.

 

4. Нажмите кнопку "Сохранить".
