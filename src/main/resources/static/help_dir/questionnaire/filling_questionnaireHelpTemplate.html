Заполнение анкеты пользователя

 

Для использования сервиса подбора меню анкета пользователя должна быть полностью заполнена.

 

1. Для заполнения анкеты пользователя войдите в личный кабинет и выберите пункт меню Анкета.

 

2. На странице Анкета выберите пункт меню Правка анкеты

 

3. На странице Правка анкеты необходимо заполнить все поля.

 

4. Нажмите кнопку "Сохранить".
