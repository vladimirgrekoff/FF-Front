Изменение пароля или email пользователя

 

При использовании сервиса у пользователя может возникнуть необходимость изменить свой пароль или адрес почтового ящика.

1. Войдите в личный кабинет и выберите пункт меню Профиль.

 

2. На странице профиля под заголовком "Профиль пользователя ..." находится строка "Изменить данные регистрации".

 

3. Нажмите кнопку "Изменить".

Далее заполните необходимые формы для изменения ваших данных.

После завершения изменения данных регистрации Вы будете переведены на Главную страницу для прохождения авторизации с новыми данными регистрации.
